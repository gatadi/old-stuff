Lab08a: Floating Lists

Prerequisites, Goals, and Outcomes

Prerequisites: Before you begin this exercise, you need mastery of the following:
  • Cascade and Inheritance
  • Altering display types (i.e. inline to block)
  • Fixing IE bugs
  • Using floats to achieve horizontal alignment of list items
  • Manipulating padding and margins to control display of background colors

Goals: Reinforce your ability to apply basic float techniques to achieve horizontal list styling

Outcomes: You will demonstrate mastery in real world application of several skills learned to date including employing floats, altering display types, applying background images & colors, and dealing with IE bugs.


Description

  1. Display the following using the given HTML, the single image images/bass_lake_beach.jpg. You do not need to modify the HTML! Only modify/create the CSS!
  2. Hint: The blue borders around each image indicate that each image is a clickable link.

     

Suggested Steps

  1. Open lab08a.html and lab08a.css and start by adding some dummy text into each of the list items. This will help you see what you're dealing with as you begin to build out your solution.  You will need to remove this text before you submit your exercise.
  2. Start by employing floats to make the list items stack horizontally instead of vertically.  Consider what difference using float: right instead of float: left would make.
  3. Next, add the background image to each anchor.  Consider what display type anchors are by default.  What are the characteristics of that display type?  Might it be necessary to alter the display type?
  4. Tackle the various background and border colors, keep in mind that background colors apply to the padding area but not the margin area.
  5. Test your solution in several browsers including IE, making sure the layout is pixel-perfect.  You may need to research any bugs you might encounter, and apply the appropriate solution.
